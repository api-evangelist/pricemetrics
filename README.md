# PriceMetrics

PriceMetrics is a price optimization and analytics platform that provides REST APIs for tracking competitive pricing data, performing elasticity analysis, and delivering revenue optimization recommendations. The platform enables businesses to integrate real-time pricing intelligence directly into their applications and workflows, supporting dynamic pricing strategies, margin analysis, and data-driven revenue management across retail, distribution, and manufacturing verticals.

## APIs

- **PriceMetrics Pricing API** — Core REST API for price optimization recommendations, competitive pricing data, elasticity analysis, and what-if scenario modeling.
